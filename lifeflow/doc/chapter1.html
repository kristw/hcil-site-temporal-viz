Chapter 1: Getting Started

Before we start

LifeFlow is a tool designed to analyze temporal event sequences by visualization. To understand more about the visualization, read the paper first to become familiar about the design of LifeFlow. The paper can be found at: http://www.cs.umd.edu/localphp/hcil/tech-reports-search.php?number=2010-22

Development environment

LifeFlow was developed in Java 1.6 using Eclipse.

Libraries

LifeFlow uses many libraries in its implementation. The three most important libraries that are used almost everywhere in the code and you should learn about to be able to understand the code are:

  • Piccolo2D : Scene graph library for Java
  • PureMVC : Model-View-Controller framework
  • MigLayout: Layout manager for Swing components

These libraries are also used, but in less places and mostly are simple calls to their APIs:

  • SLF4J : Wrapper for logging libraries
  • Apache log4j : Logging library
  • SwingX : Custom Swing Components
  • Flamingo : Custom Swing Components
  • MacWidgets : Swing components library that makes components look "Mac"
  • Apache Commons-Math : Math and statistics library

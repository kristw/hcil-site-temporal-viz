1. How to change the placement of events from absolute time to relative time with respect to one specific event, e.g. ICU?

    2. Align the records by that event (ICU) and the rest of the events will be displayed by their relative time to that event accordingly. (4.6 Alignment)

  2. In the Overview Panel, how to set the event order precedence for events that happen at the same time?

    3. The events that happen at the same time will be sorted in the order of the event types in the legend panel. You can drag the event types in the legend up and down to customize. (4.1.4 Change the order of event types)

Display mode

Normally the X-axis is used to represent time. After unchecking the X-axis checkbox, the elapse time of all the events are treated equally. This can be useful when only the sequence of events is of concerned but not the time between events.

The default use of the Y-axis is to represent the number of records. After unchecking the Y-axis checkbox, LifeFlow displays all the sequences with equal height. This can be useful when the users want to look at some sequences having a small amount of records.

Sort records in distribution

When showing the distribution, the default setting is to arrange the records in ascending order of the time gap between the two events. An alternative is to arrange the records in ascending order of the time of day of the start event. An example of applying this alternative is to find the possible tendency of the distribution with respect to the time of the start event.

Unchecking "Show distribution on mouse over" disables the distribution feature. Checking "Group distribution by next event" group the distribution by the sequence of events. This can help the users to see the difference in the distribution with respect to the sequence.

Episode beginning category

If one of the event is considered to be the episode event, the users can click the radio button and select that event in the pull-down menu. LifeFlow will draw a dotted line between the last preceding event before the episode event to mark the beginning of new episode.

Representative time

The default criterion to aggregate the time is mean. In some special cases the users may want to switch from mean to other criteria, such as medium, max, min, or standard deviation.

Repeated events can be merged by checking the checkbox "Merge repeated events". After checking "Fill gap with previous event's color", the color of the preceding event will be used to fill the gap.
